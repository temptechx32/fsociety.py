Webring Rules

  • Your website must fit the Webcore aesthetic: retro, web 2.0, Y2K, etc.
  • No NSFW content.
  • No hate speech or discrimination.
  • Include the webring widget on your site.

Join the Webring

To join, email aeroxptech@gmail.com with your website URL and ensure you have the widget installed.

Navigation Widget

Include this code on your website:

<div id="webcore-webring" class="webcore-webring-js">
  <script src="https://aeroxptech.github.io/fsociety.py/assets/webassets/assets/javascript/webbring.js"></script>
</div>

<noscript>
  <style>
  /* Hide JavaScript Widget */
  .webcore-webring-js {
    display: none !important;
  }
  </style>

  <!-- No JS Widget -->
  <div id="webcore-webring">
    <div style="display: flex; flex-direction: column; align-items: center; font-size: 1rem;">
      <a href="https://aeroxptech.github.io/fsociety.py/assets/webassets/othertabs/webbring.html" target="_top">Webcore Webring</a>
      <div style="display: flex; gap: 1rem; font-size: 1.5rem">
        <a href="https://aeroxptech.github.io/fsociety.py/assets/webassets/othertabs/webbring.html#members" target="_top">No JavaScript</a>

</noscript>
